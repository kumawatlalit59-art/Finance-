📊 Financial Data Analysis & Forecasting Dashboard
📌 Project Overview

This project focuses on analyzing a company's financial performance using SQL, Python, and Power BI. The objective is to transform raw financial data into meaningful business insights through data cleaning, analysis, forecasting, and interactive visualizations.

The project demonstrates an end-to-end data analytics workflow, from data extraction and preprocessing to dashboard creation and financial forecasting.

🎯 Objectives
Analyze company financial performance.
Identify revenue and profit trends over time.
Compare business unit and regional performance.
Forecast future financial metrics using time series analysis.
Build an interactive Power BI dashboard for decision-making.
🛠️ Tools & Technologies
SQL Server – Data storage and SQL queries
Python – Data cleaning and forecasting
Pandas – Data manipulation
Matplotlib – Data visualization
Statsmodels – Time Series Forecasting
Power BI – Interactive Dashboard
Microsoft Excel – Dataset Management

The dataset contains financial records with the following fields:

Business Unit
🔄 Project Workflow
1. Data Collection
Imported financial dataset into SQL Server.

Created database and table.

3. Python
Data cleaning

Duplicate removal
Date formatting
Financial forecasting using Holt-Winters/ARIMA

Developed an interactive dashboard containing:

Profit by Region
